/api/send_recording

URL: /api/send_recording

Type: POST

Status Codes: 201 Created, 400 Bad Request

Example Request

{
    "sensorid": 74464,
    "timestamp": datetime.utcnow().isoformat(),
    "ts_data": [1.00,2.00,3.00]
}

Example Response

{
    "message": "Data added successfully"
}

/api/add_sensorconfig

URL: /api/add_sensorconfig

Type: POST

Status Codes: 201 Created, 400 Bad Request

Example Request

{
    "model": "ExampleModel",
    "userid": 1,
    "floor": "ExampleFloor",
    "wall_radius": 2.5,
    "obstacle_radius": 1.0
}

Example Response

{
    "message": "Sensor record created",
    "sensorid": "74464"
}
